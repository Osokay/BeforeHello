在为深圳某客户提供研发管理平台实施和培训的项目中[1]，应客户要求提供了两套问卷试题。背负考核任务去学习，团队转换配置管理工具的效果自然会更好吧。

问卷A最早公布于我的博客，并由@非专业IT民工给出了答案和详尽的解释：

  • http://loveky2012.blogspot.jp/2012/08/git-knowledge-test-answer.html[2]

问卷B和问卷A的相似度非常高，就请保留一些神秘性吧，好让问卷B能够承担一些自测功能。尝试解答问卷B是毫无必要和没有技术含量的。

BTW，问卷的设计是在北京到深圳的飞机上匆匆完成的，对我来说为错误答案而凑数是最花脑筋的，不严谨之处敬请谅解。
